# Overview
This is a simple game tracker for the player playing as the empire in the Star Wars board game [Rebellion](https://a.co/d/00Z1sJfB). It will track if you have probe cards, troops and loytaly in each system. It will tell you which systems you still have to check and which regions are good, enjoy!

# Install
Requirement if you dont have python:
```
brew install python
```
First:
```
git clone https://github.com/calebzaleski/Rebellion-Empire-Tracker.git
```
Then:
```
cd ~/Rebellion-Empire-Tracker
```
Finally:
```
python3 main.py
```
Have fun!
